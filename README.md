# NUTRICODE - SPRINT2
## Requisitos: Docker 20+ / Docker Compose e Node.js 18+
1.Clone o repositório no terminal: 
git clone https://github.com/SulamitaViegas123/nutricode-devops-sprint2.git
cd nutricode-devops-sprint2

2.Agora deve subir containers:
docker compose up -d --build

3. Verificar se está rodando:
No navegador: http://localhost:3000
ou no terminal: docker ps

## Observações
- Container rodando em background  
- Diretório padrão do app: `app/`
- Usuário sem privilégios administrativos pode executar
